# Build a Fullstack Inventory Management Dashboard

This project is a complete full-stack inventory management dashboard built using modern web technologies and deployed on AWS. The frontend is developed with Next.js and styled using Tailwind CSS, featuring advanced data handling through Material UI Data Grid. State management is implemented with Redux Toolkit, while Redux Toolkit Query is used for efficient data fetching and caching.

The backend is powered by Node.js, with Prisma ORM handling database operations and ensuring scalable, maintainable data access.

The application is fully integrated with AWS services, including Amazon RDS for database management, EC2 for backend hosting, API Gateway for building and managing APIs, S3 for file storage, and Amplify for frontend deployment.

## Tech Stack

- **Next JS**
- **Tailwind**
- **Redux Toolkit**
- **Redux Toolkit Query**
- **Material UI Data Grid**
- **Node.js**
- **Prisma**
- **AWS EC2**
- **AWS RDS**
- **AWS API Gateway**
- **AWS Amplify**
- **AWS S3**
